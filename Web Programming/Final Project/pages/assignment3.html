Models

"selectOption" is number selection(1-9) that user picks.

"boardTable" is a reference to the board table element. This will be used to update the table element once it's created.

"puzzle" This is used to create the initial board.

"solution" This is puzzle's solution, this is used to check if user input is correct.

Functions/Methods

createGameSelectOptions():

This creates the button group for the number selection(1-9), selection will be stored in selectOption variable.

createGameBoard(rows, cols):

This creates a table element with given rows and columns.

setupInitialBoard(board):

Initial board setup with puzzle's given numbers highlighted and for the remaining user numbers onclick handlers created.

getSelectedDifficulty():

Checks the radio buttons that selects the difficulty.

getBoardValues(difficulty):

Create a board with given difficulty. Puzzle for the board will be selected randomly.

startNewGame():

Starts a new game. Calls all the functions needed to start the game.

main():

Creates the html elements once window is loaded.
