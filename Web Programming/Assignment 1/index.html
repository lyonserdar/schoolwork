Assignment 1: Project Description and Game Grid

Description Page

For this project I picked the Adventure/dungeon game map.

The game terrain will have a tile map and the player will have a tile based movement using keyboard keys(W, A, S, D or arrow keys).

The map will have border tiles that are impassable, and some other tiles like walls also impassable

If the player get to the destination tile, the game will end. The destination tile will be protected by a locked gate.

Player will have to obtained the key to unlocked the gate.

The game will have a basic inventory to hold the key. And display it to the user once they obtain it.
